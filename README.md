# MoOps — AI 驱动的全平台增长运营管理桌面客户端

> 一站式增长运营中枢：覆盖海外社媒、国内社媒（小红书 / 抖音）、Amazon、独立站（DTC），用 AI 把内容生产、数据分析、广告优化、诊断建议统一到一个工作台。

## ✨ 功能全景

### 📊 数据看板
全局运营概览，跨平台核心指标一屏掌握。

### 📱 海外社媒运营
| 模块 | 能力 |
|------|------|
| AI 内容工厂 | 多平台文案生成、手动输入目标受众 + AI 推荐细分群体 |
| 内容排期日历 | 选题 / 节日 / 热点排期 |
| 社媒数据分析 | 多平台指标可视化 + **AI 运营诊断** |
| Hashtag 研究 | 标签挖掘与组合建议 |
| A/B 测试 | 多版本文案对比实验 |

### 🇨🇳 国内社媒运营
**小红书运营**
- 笔记内容工厂：选题 → 标题 → 正文 → 封面文案 → 标签
- 经营月报 OCR：**上传月报截图 → AI 识别数据 → 可视化图表 + 诊断建议**
- 爆款笔记拆解：输入爆款 → AI 分析结构 → 可复用模板
- 关键词 SEO：搜索词挖掘 + 埋词建议

**抖音运营**
- 视频脚本工厂：钩子 + 正文 + CTA + 分镜口播稿
- 直播话术库：开场 / 逼单 / 互动话术分类管理
- 直播流程策划：按产品 + 时长生成直播流程表
- 视频数据分析：播放 / 完播 / 互动可视化 + AI 诊断

### 🛒 Amazon 运营（一级导航）
- 运营总览、广告 CSV 分析（ACoS / CTR / CVR + **AI 投放诊断**）、评论分析、Listing 优化、利润计算器

### 🌐 独立站运营（DTC）
- 广告投放分析、SEO 内容引擎、转化优化（Landing Page 诊断）、邮件营销工厂、独立站数据看板

### 📅 周行动计划 / 📝 工作报告
- 周行动计划：自动排期与跟进
- 工作报告：日报 / 周报 / 月报，AI 流式生成，历史保存

## 🌗 主题
全站支持 **白天 / 黑夜主题切换，默认黑夜模式**，右上角一键切换，选择自动记忆。

## 📦 下载与安装
1. 前往 [Releases](https://github.com/YuyuetMo/MoOps/releases) 下载 `MoOps-vX.Y.Z-win.zip`
2. 解压后会得到一个 `MoOps-vX.Y.Z` 文件夹
3. 运行其中的 `MoOps.exe` 即可启动
4. 应用内置自动更新，打开时会自动检测新版本

> 需要 AI 功能时，在设置中填入 API Key（支持 OpenAI / DeepSeek / 小米 MIMO / Claude 等）。
> 图片识别类功能（如小红书经营月报 OCR）请使用支持视觉的模型（如 gpt-4o / mimo-v2.5）。
